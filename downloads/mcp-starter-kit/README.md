# MCP Starter Kit

Minimal downloadable starter kit for an MCP-style docs tool.

## Included
- `package.json`
- `server.js`
- `.env.example`

## Purpose
Give buyers a fixed starting point for local experiments with docs lookup and a
simple JSON tool endpoint.

## Rules
- This is a starter kit, not a custom build.
- Modify locally after download for your own use case.
